# Para limpiar la caché y ver los cambios:

## 1. En tu máquina local:
```bash
# Detén el servidor de desarrollo (Ctrl+C)

# Limpia la caché de node_modules y rebuild
rm -rf node_modules/.vite
rm -rf dist

# Reinicia el servidor
npm run dev
```

## 2. En el navegador:
- Presiona Ctrl+Shift+R (Windows/Linux) o Cmd+Shift+R (Mac) para hard refresh
- O abre DevTools (F12) > Network tab > marca "Disable cache" > recarga
- O prueba en modo incógnito

## 3. Si estás viendo la versión de producción (Vercel):
- Espera a que Vercel termine el deploy del último commit (332f762)
- O fuerza un nuevo deploy desde el dashboard de Vercel
